# Experiments

This folder defines how experiments should be specified and logged.

## Conventions
- Create one experiment spec from `TEMPLATE.md`
- Use descriptive IDs, e.g.:
  - `exp_2026-04-02_baseline-mpc-v1`
- Keep assumptions and configuration details explicit

## Required Metadata
- owner
- run date/time
- simulator/environment version
- code commit hash
- config summary
- metrics tracked
- artifact locations
