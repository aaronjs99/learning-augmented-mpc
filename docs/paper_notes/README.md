# Paper Notes Workflow

Use this folder to capture reusable, implementation-relevant notes from individual papers.

## How to Use
1. Copy `TEMPLATE.md` to a new file named:
   - `YYYY-MM-DD_short-paper-key.md`
2. Fill in all required sections, especially assumptions and implementation relevance
3. Add fusion opportunities that connect this paper to at least one other candidate
4. Link the note in any related issue/PR

## Goal
These notes should support paper selection, implementation planning, and transparent research decisions.
